Panel with tabs in the heading

Panel title with tabs
  • Usage
  • Sample Code
 
<div class="panel-heading" >
    <div class="panel-title" >
        <span class="lsi-panel-title-tabs">Panel title with tabs
            <ul class="nav nav-tabs">
                <li class="active" ><a href="#usage" data-toggle="tab">Usage</a></li>
                <li><a href="#code" data-toggle="tab">Code</a></li>
            </ul>
        </span>
    </div>
</div>
Use the lsi-panel-title-tabs class in the heading of the panel to add tabs.
See the code example in the "Code" tab.

Lobstered font

Use the lsi-lobstered class to have the text rendered with the Lobster font.

If the lsi-lobstered class is used in a well, the text color will be changed to gray.

Main action button

The lsi-call-action class will make the text inside the button upper case and add a small shadow.
